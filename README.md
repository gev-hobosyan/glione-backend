# Glione Backend

## What is this?
Glione Backend, as name might suggest is the backend part of [Glione](https://github.com/gev-hobosyan/glione). It handles the creation and serving of the lessons, as well as user stats, such as: XP, Eneregy and etc.

## Tech Stack
- **Framework** - [Hono](https://hono.dev/)
- **Database** - [MongoDB](https://www.mongodb.com/)
- **Hosting** - [Vercel](https://vercel.com/)
